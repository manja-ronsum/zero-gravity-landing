# Zero Gravity Landing 🚀

A next-generation, mobile-first landing page concept that breaks free from heavy frameworks and clunky libraries. Experience a smooth, weightless scrolling experience powered entirely by modern web standards.

## 🌟 Why "Zero Gravity"?

Traditional landing pages are often weighed down by heavy dependencies like jQuery or bloated carousel plugins. **Zero Gravity Landing** is different. It uses **native CSS Scroll Snapping** to deliver a frictionless, native-app-like swipe experience directly in the browser—with zero external libraries.

## ✨ Features

- **100% Vanilla Tech Stack:** Built purely with HTML, CSS, and minimal JavaScript.
- **Native Scroll Snapping:** Buttery smooth, zero-latency carousel transitions using `scroll-snap-type`.
- **Mobile Perfected:** Designed specifically for mobile viewports using `100dvh` for a pixel-perfect full-screen experience.
- **Zero Dependencies:** No `npm`, no build steps, no jQuery. Just pure, unadulterated performance.

## 🚀 Getting Started

Simply clone the repository and open `index.html` in your browser. No build process required!

```bash
git clone https://github.com/yourusername/zero-gravity-landing.git
cd zero-gravity-landing
open index.html
```

## 🛠 Tech Stack
- HTML5
- CSS3 (Scroll Snapping, Variables, Gradients)
- Vanilla JavaScript (IntersectionObserver for micro-animations)
